    • Admyral

    Admyral is a minimal keyboard driven web browser written in Python and Qt.

    Keyboards Shortcuts

    • F2: Open Tabs List
    • F6: Open URL Bar

    • Ctrl+J: Previous Tab
    • Ctrl+K: Next Tab
    • Ctrl+L: Open URL Bar
    • Ctrl+T: Open New Tab
    • Ctrl+W: Close Tab
    • Ctrl+Q: Close Browser
